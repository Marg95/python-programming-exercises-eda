# Python Programming Exercises

## Project Description
This repository contains a Jupyter Notebook with Python exercises developed as part of practical training in programming and problem-solving. The notebook focuses on applying core Python concepts through guided exercises and examples.

## Objectives
- Practice fundamental Python programming concepts
- Improve logical reasoning and problem-solving skills
- Gain familiarity with Jupyter Notebooks
- Build a foundation for data analysis and scientific computing

## Technologies
- Python 3
- Jupyter Notebook
